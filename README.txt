****************** INSTALLATION *****************

Nainstalujte knihovnu OGDF www.ogdf.net
Nastavte cestu ke knihovné OGDF v souboru gal.config

gcc Compiler (Linux, Mac OS):

 A) Pomocí aplikace Qt Creator

 B) Pomocí qMake
  1. Ve složce aplikace spustíme qmake pro vygenerování Makefile

  2. Pomocí make sestavíme aplikaci

Visual Studio (Windows):

  1. Nainstalujte doplnìk Visual Studio Add-in 1.2.4 for Qt5

  2. Otevøete projekt pomocí GAL.pro a sestavte.
